# gengine


Introduction
============
It is engine based on OpenGL and Qt to develop 3D Graphics on desktop.                                                                              

Requirements
============
- OpenGL : over 2.1
- Qt : 5.10.1


Example
=======
To use examples, gandis 3D engine must be built on QT and OpenGL environment. Each example shows on 'SampleWindow' like as below.
